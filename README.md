# 📉 Customer Churn Prediction

## 📌 Overview
Predicting customer churn using Machine Learning to help telecommunications companies identify at-risk customers.

## 🛠️ Features
- **Data Analysis:** Exploratory Data Analysis (EDA) & Visualization.
- **Modeling:** Compared Logistic Regression, Random Forest, and XGBoost.
- **Web App:** Interactive dashboard built with Streamlit for real-time predictions.

## 📊 Performance
- **Model:** XGBoost / Random Forest.
- **Recall:** ~79% (Focusing on capturing potential churners).
